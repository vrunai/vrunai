# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-26

### Added
- Agent Definition Language (ADL) — YAML-based spec format for defining agent behavior
- Path accuracy, tool accuracy, and outcome accuracy evaluation
- Consistency scoring across multiple runs
- Multi-provider support: OpenAI, Anthropic, Google, xAI, DeepSeek, Mistral, Ollama
- Cost tracking with 26 built-in models and pricing
- Interactive TUI (Ink/React)
- Web UI (React/Vite)
- Model catalog with custom model configuration
- 6 example evaluation specs (customer support, expense approval, HR onboarding, IT helpdesk, invoice processing, security incident)
